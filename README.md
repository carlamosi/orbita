# 🌍 Orbita
### Master the world with high-intensity geography challenges.

**Orbita** is a high-performance, immersive geography platform designed to turn world discovery into a cinematic experience. Built for people who value both precision and aesthetic, it combines a fluid 3D engine with deep gamification loops to make global mastery feel less like study and more like a voyage through the stars.

---

## ✨ The Experience

Orbita is built on three core pillars: **Motion, Feedback, and Data.**

*   **Cinematic 3D Interface**: A high-fidelity globe powered by `react-globe.gl` and `Three.js` that reacts to your every move.
*   **High-Intensity Game Modes**: From 60-second **Speed Rounds** to pixel-perfect **Flag Quizzes** and **Capital Mastery**.
*   **Intelligent Heatmapping**: Your progress isn't just a number. Watch the world light up in neon green and violet as you master countries, showing your global resonance in real-time.
*   **Arcade-Grade Feedback**: Spatial audio triggers and 1-frame haptic flashes ensure every correct answer feels rewarding and every wrong one feels like a lesson.

---

## 🛠️ The Modern Stack

Orbita is a **local-first** application, meaning your data stays with you, in your browser, running at a buttery-smooth 60 FPS.

*   **Core Logic**: React + Vite (Ultra-fast HMR and build performance)
*   **State & Persistence**: Zustand with persistent storage (your progress survived refreshes)
*   **Motion Architecture**: Framer Motion (cinematic page transitions and micro-interactions)
*   **3D Engine**: Three.js / React-Globe.gl (GPU-accelerated world rendering)
*   **Analytics**: D3.js and D3-Geo (precision heatmap math and geographic projections)
*   **PWA Ready**: Offline-first architecture with intelligent Service Worker caching for assets and flag data.

## 🚀 Getting Started

To launch your own instance of Orbita, follow these simple steps:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/carlamosi/Orbita.git
    ```
2.  **Install dependencies**
    ```bash
    npm install
    ```
3.  **Start the engines**
    ```bash
    npm run dev
    ```
